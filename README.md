# Language Translation Tool

A simple web application built using:

- HTML
- CSS
- JavaScript
- Flask
- LibreTranslate API

Features:
- Enter text
- Select source language
- Translate
- Display output

Run locally:

pip install -r requirements.txt

python api.py
